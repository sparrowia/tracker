Read these files to understand the Edcetera PM Tracker before working on it:

1. `PROJECT.md` — full project overview, stack, database schema, features, UI system, deployment
2. `CLAUDE.md` — coding conventions, guardrails, multi-project environment warnings
3. `src/lib/types.ts` — all TypeScript interfaces and enums
4. `src/lib/utils.ts` — formatting helpers (priority colors, status badges, dates)

For database context, read the relevant migration in `supabase/migrations/`. For component context, read the relevant file in `src/components/`.
